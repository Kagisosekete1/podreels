
# PodReels - Full App Transformation Plan

## Phase 1: Foundation (This session)
1. **Enable Lovable Cloud** - Set up database, auth, storage
2. **Database schema** - profiles, reels, likes, comments, follows tables
3. **Design system** - Bold & vibrant theme with podcast-focused colors
4. **Core layout** - Mobile-first vertical feed layout with bottom navigation

## Phase 2: Core Pages & Components
5. **Auth page** - Sign up/Sign in with viewer/podcaster roles
6. **Vertical Feed** - Swipeable full-screen PodReels player (2min+ clips)
7. **Video Player** - Play/pause, mute, progress bar, auto-advance
8. **Interaction buttons** - Like, comment, share overlays

## Phase 3: Creator Features
9. **Upload flow** - Upload PodReels with title, description, podcast name
10. **Creator profiles** - Profile page with uploaded reels grid
11. **Storage bucket** - For video uploads and avatars

## Phase 4: Social Features
12. **Comments** - Comment modal on each reel
13. **Follow system** - Follow podcasters
14. **Discover/Explore** - Browse by podcast category

### Key decisions:
- Min video length: 2 minutes
- Vertical format (9:16 aspect ratio)
- Categories: Comedy, True Crime, Tech, Business, Health, Education, News, etc.
- Roles: Viewer and Podcaster (creator)
